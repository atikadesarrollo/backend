🔗 Enviar a Odoo Online

📋 Pasos previos en Odoo Online:
1. Ir a Configuración → Usuarios → [su usuario] → Seguridad de cuenta
2. Clic en "Nueva clave API" → copiar la clave
3. El campo Base de datos es el subdominio: su-empresa de su-empresa.odoo.com
✅ Método recomendado para Odoo Online (evita CORS)
Make.com (gratis) actúa como puente entre el cotizador y Odoo. Sin instalar nada en el servidor.
Cómo configurar en Make.com (5 pasos):
  1. Crear cuenta gratis en make.com
  2. Crear un Escenario nuevo → agregar módulo Webhooks → Custom Webhook
  3. Copiar la URL del webhook que entrega Make
  4. Agregar módulo Odoo → Create a Record → conectar su cuenta Odoo
  5. Mapear los campos: partner_id, note, order_line
Exportar la cotización como JSON para importar manualmente en Odoo o compartir con su equipo técnico.
